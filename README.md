# NITT Innovation Challenge — Alumni Connect 🚀

Welcome to the **IIC-demo-website**, the official registration portal for the NIT Trichy Innovation Challenge, organized by the Institute Innovation Council (IIC).

### About The Project
This platform is designed to gather student perspectives on alumni engagement at NIT Trichy. It features:
- A modern, responsive, and dynamic UI using HTML/CSS and Vanilla JavaScript.
- A custom infinite scrolling gallery (no React required).
- Strict form validation to ensure accurate data collection (NITT emails, 9-digit roll numbers, etc.).
- A serverless backend API ready for Vercel deployment.

### Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend/Deployment**: Vercel Serverless Functions (Node.js)

### Deployment
This project is fully configured to be deployed directly to Vercel with zero additional setup. Simply push to GitHub and link your Vercel account.
